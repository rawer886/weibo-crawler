# Scripts 目录

该目录包含用于查询和分析已抓取数据的脚本。

## 可用脚本

### query_blogger_comments.py
查询特定微博下博主自己的评论。

**用法：**
```bash
python scripts/query_blogger_comments.py <微博ID>
```

**示例：**
```bash
python scripts/query_blogger_comments.py 5253489136775271
```

**输出示例：**
```
【评论 1】
  评论ID: 5253489136775271_6329764758701369623
  时间: 26-1-10 13:51
  点赞数: 41
  📝 评论内容: 出清还要死一批
  ↳ @不出轨了: 有我一个ST归零的奉献
```

**说明：**
- 显示微博基本信息（博主、发布时间、内容摘要）
- 列出博主在该微博下的所有评论
- 如果是回复其他评论，会显示被回复者和被回复内容
- 如果是直接评论微博，不显示引用信息
