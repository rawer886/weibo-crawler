# Tests - 功能测试脚本

该目录包含用于测试和调试爬虫功能的测试脚本。

## 目录说明

tests 目录提供各种测试工具，用于：
- 测试单个功能模块
- 调试爬虫逻辑
- 验证数据正确性
- 开发和调试新功能

## 可用测试

### test_single_post.py

**功能**: 测试单条微博的评论抓取功能

**用途**:
- 快速测试特定微博的评论抓取
- 调试评论解析逻辑
- 验证回复关系识别
- 开发新功能时的快速验证

**用法**:
```bash
# 1. 激活虚拟环境
source venv/bin/activate

# 2. 运行测试
python tests/test_single_post.py <博主UID> <微博ID>
```

**注意事项**:
- ⚠️ 需要先激活虚拟环境
- 🌐 会打开浏览器窗口（非无头模式）
- 🔐 首次运行需要手动登录
- 📊 测试数据会保存到数据库


## 测试最佳实践

1. **隔离测试**: 每个测试脚本专注测试一个功能
2. **清理数据**: 测试前清理相关数据，避免干扰
3. **详细日志**: 输出详细的调试信息
4. **错误处理**: 妥善处理异常，确保浏览器正常关闭
